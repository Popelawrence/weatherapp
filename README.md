# Weather Information App

## Overview
The Weather Information App is a JavaFX-based desktop application that provides real-time weather updates for any city.

!Screenshot of Program Output


## Features
- Real-Time Weather Updates
- Dynamic Icons
- Unit Conversion
- History Tracking
- 5-Day Forecast
- Current Time Display
- User-Friendly GUI
- Error Handling

## Technologies Used
- Java
- JavaFX
- OpenWeatherMap API
- JSON Parsing
- HTTP Requests

## Setup Instructions
- Install Java JDK 17 or later.
- Download the JavaFX SDK.
- Obtain an API key from OpenWeatherMap.
- Follow the installation steps to set up the project.

## How to Use
- Launch the application.
- Enter a city name and click "Get Weather" or press "Enter".
- View real-time weather details and the current time in the respective city.
- Check the search history for recently queried locations.

## Error Handling
- Handles invalid location inputs or failed API requests gracefully.

## Future Enhancements
- Implement geolocation-based weather updates.
- Include more detailed metrics like air pressure and visibility.
- Allow users to toggle between metric and imperial units for wind speed.
